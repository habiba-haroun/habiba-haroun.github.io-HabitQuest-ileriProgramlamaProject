HabitQuest – Study Tracker

HabitQuest, kullanıcıların günlük çalışma görevlerini oyunlaştırma yaklaşımıyla takip etmelerini sağlayan tarayıcı tabanlı bir web uygulamasıdır. Kullanıcılar görevlerini tamamladıkça XP, seviye ve coin kazanır, günlük streak bilgilerini takip eder ve belirli bir seviyeden sonra mağazayı açarak sanal ürünler satın alabilir. Uygulama tamamen HTML, CSS ve saf JavaScript kullanılarak geliştirilmiş . Tüm veriler LocalStorage üzerinden saklanır ve proje İleri Programlama dersi kapsamında JavaScript, DOM yönetimi ve veri saklama konularını pratik olarak göstermektedir.
